# View Management

Control your drawing view and zoom.

## Zoom Operations

```python
# Zoom to extents (see all objects)
cad.zoom_extents()

# Zoom to specific object
cad.zoom_to_object(circle)
```

## Pan and Navigation

```python
# Pan to a point
cad.pan(APoint(10, 10, 0))
```
